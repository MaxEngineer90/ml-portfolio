# **App Name**: DevFolio Pro

## Core Features:

- Project Showcase: Showcase developer projects in a gallery layout.
- About & Skills Section: Detail developer skills, experience, and personal summary.
- Contact Form: Enable potential clients/employers to contact the developer.
- AI Intro Message: Use an AI tool to personalize introductory message based on visitor's LinkedIn profile.

## Style Guidelines:

- Primary color: Deep orange (#FF4500) to give a sophisticated techy feeling.
- Background color: Dark gray (#212121) for a modern, professional look.
- Accent color: Teal (#00BCD4) to provide a techy highlight and contrast.
- Body font: 'Inter', sans-serif, for a modern and readable feel.
- Headline font: 'Space Grotesk', sans-serif, for a bold, tech-inspired look. Use 'Inter' for longer text.
- Use minimalist, line-style icons for skills and categories.
- Subtle transitions and animations to improve user experience.
